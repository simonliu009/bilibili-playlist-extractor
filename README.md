这是一个 Chrome 扩展，结合了两个功能：

1. 在 B 站合辑视频页面右键选择“导出视频信息为Markdown”，会将合辑的视频信息复制为 Markdown 格式。
2. 在任意网站页面双击鼠标中键，会复制当前页面标题和链接为 Markdown 格式；三击鼠标中键则仅复制当前 URL。

扩展会自动清洗 URL 中的推广参数，并提供可选设置控制是否显示详细复制内容。

在扩展选项页里可以切换：

- 是否显示详细复制内容
- 是否清理 URL 中的推广参数

如果你修改了图标源文件，可以运行 `node generate-icons.js` 重新生成 `icon16.png`、`icon48.png` 和 `icon128.png`。

本扩展最初使用Cursor AI开发而成，后续迭代改用codex。
